Dual-Issue Sony Cell Synergistic Processor Unit (SPU)

Superscalar RISC Core Implementation in SystemVerilog

Overview

This project involves the architecture and RTL implementation of a 2-way superscalar RISC core based on the IBM/Sony Cell Broadband Engine SPU ISA. Designed for high-performance computing, the core features a 128-bit SIMD data path and a deep pipeline structure to maximize throughput.

Architecture

The processor implements a heterogeneous dual-issue pipeline where instructions are dispatched to either the Even (Arithmetic/Logic) or Odd (Load/Store/Permute) pipe.

  • Pipeline Depth: 11 stages to support high frequency operation.
  • Register File: 128-entry unified register file (128-bit wide) with 6 read ports and 2 write ports.
  • Local Store: 32KB high-speed local memory for instruction and data storage.
  • Hazard Handling: Advanced hazard detection logic handles dual-instruction dependency checking and extensive operand forwarding networks minimize stalls.

Toolchain & Verification

To validate the hardware implementation, a custom software toolchain was developed.

  • Custom Assembler: Developed in C++ to map SPU Assembly mnemonics to machine code.
  • Simulation: Full assembly programs are simulated in Siemens QuestaSim to verify architectural correctness.

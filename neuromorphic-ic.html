Low-Power IC for Neuromorphic Computing

Project Overview

This project involves the design and verification of ReRAM, a compute-in-memory (CIM) chip based on Resistive Random-Access Memory (RRAM). The architecture addresses the "memory wall" bottleneck in edge AI by performing computation directly within memory arrays, eliminating power-hungry data movement.

1. Architecture: Transposable Neurosynaptic Array (TNSA)

Conventional RRAM-CIM architectures are often limited to fixed dataflow directions. To enable versatility for diverse AI models (CNNs, LSTMs, RBMs), we implemented a Transposable Neurosynaptic Array (TNSA).

The TNSA consists of 256x256 RRAM cells (768 CMOS Neurons and 196,608 ReRAM cells total).

2. Measured Results

The design was validated through software simulation using the SpikingJelly framework for Spiking Neural Networks (SNNs). We achieved high accuracy on keyword spotting tasks, demonstrating the effectiveness of our proposed architecture.

  • Google Speech Command: 83.12% Accuracy (8-Bit Quantized SNN Simulation)
